Units (papersizes.units)¶

Units and unit conversions used in the papersizes module.

papersizes.units.cm = 28.34645669291339¶

One ISO-standard centimeter.

papersizes.units.inch = 72.0¶

One International inch (defined as 25.4mm).

papersizes.units.m = 2834.645669291339¶

One ISO-standard meter.

papersizes.units.mm = 2.834645669291339¶

One ISO-standard millimeter.

papersizes.units.pt = 1.0¶

One postscript point.

Conversions¶

This module also defines conversions between the above units, e.g.:

>>> 25.4 * mm2inch
1.0
>>> 20 * cm2m
0.2
>>> inch
72
>>> inch2pt
72
